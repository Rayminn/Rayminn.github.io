# 关于Github学生包

Github学生包里包括了一大揽子内容，本人目前用到的有`Github Pro`，`Github Copilot`，`Jetbrains 全家桶`等。

具体内容可见[Github 学生包](https://education.github.com/)页面。

# 申请指北

## 申请条件

- 年满13周岁
- 文凭教育在读
- 能够开具证明

## 申请准备

使用`MS Edge`，`Chrome`，`Safari`，`Firefox`中的一个浏览器。

填写`Github`账户信息，确保信息中姓名与证明的证明相同。

开启`2fa`双重认证。

搞到带学校名称与本人名称的证明（学信网、学生证或者充分运用人类智慧）

> 本人的证明是用成绩证明截掉成绩爆改而来（）

## 申请过程

![pic1](/blogpic/github-edu1.png)

选择用于认证的邮箱，有`edu`邮箱的最好用`edu`邮箱，但本人作为苦x高中生甚至学校没有`edu`网站，但普通邮箱用正式证明也能过认证。

![pic2](/blogpic/github-edu2.png)

填写学校名称，对于大陆学校输入中文全名就会出现包含中文名英文名与学校网站的选项。

填写你申请认证的理由，但这个理由并不重要，本人只写了下面一句话：

> I wanna learn to code and try to make contribution to the community.

然后将你准备的证明上传到网站上，并等待五天左右就大功告成了！

![pic3](/blogpic/github-edu3.png)

申请生效后侧栏会出现如图申请记录，在成功与生效直接还需要等待三天左右。成功后`Approved`是绿色的，此时不会有邮件通知，生效后会变成紫色并邮件通知，然后就可以使用学生包内的所有内容啦！
